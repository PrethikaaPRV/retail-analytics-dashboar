# 🛒 Retail Analytics Dashboard

## 📌 Project Overview
An end-to-end retail analytics project analyzing 9,800+ sales records across 4 regions and 3 product categories using Python, SQL, and Power BI.

## 🛠️ Tools & Technologies
- **Python** — Pandas, Matplotlib, Seaborn (EDA & visualization)
- **MySQL** — Data storage & SQL analysis queries
- **Power BI** — Interactive dashboard with slicers and KPI cards

## 📊 Dataset
- **Source:** Superstore Sales Dataset (Kaggle)
- **Size:** 9,800 rows × 18 columns
- **Period:** 2015–2018

## 🔍 Key Insights
- 💰 **Total Revenue:** $2.26 Million across 4 years
- 🏆 **Top Region:** West ($710K) — 31% of total sales
- 📦 **Top Category:** Technology ($827K)
- 👥 **Largest Segment:** Consumer (52% of all orders)
- 📈 **Growth Trend:** Sales grew consistently from 2015 to 2018
